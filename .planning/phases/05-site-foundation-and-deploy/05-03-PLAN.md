  <action>
1. Use the GitHub CLI to enable GitHub Pages with GitHub Actions as the source:
   ```bash
   gh api repos/stigsb/matlab-course/pages -X POST -f build_type=workflow 2>/dev/null || gh api repos/stigsb/matlab-course/pages -X PUT -f build_type=workflow
   ```
   If this fails with a permission error, create a checkpoint for the user to enable it manually in Settings > Pages > Source: GitHub Actions.

2. Push the current main branch to trigger the deploy workflow:
   ```bash
   git push origin main
   ```

   ```bash
   gh run list --workflow=deploy.yml --limit=1
   ```
   Monitor until complete. If it fails, diagnose from the logs.
