**01-first-steps/lesson.md** — The written lesson for Module 01. Structure:
1. Opening: "Welcome! By the end of this lesson you will have written and run your first Matlab script." Keep warm, encouraging, concise.
2. Section: "The Matlab Desktop" — Brief orientation (Command Window, Editor, Workspace, Current Folder). 2-3 sentences each with what they do. Tell learner to open Matlab and identify these panels.
3. Section: "Your First Script" — Walk through creating demo_hello.m. Explain: open Editor, type code, save, press F5. Reference demo_hello.m by name.
4. Section: "Variables — Giving Names to Things" — Explain variables using Christel's life: `enzo_weight = 5.2` (cat), `ravn_age = 11`, `helena_age = 8`. Cover scalars (numbers), strings (`name = 'Enzo'`), booleans (`is_cat = true`). Show `disp()` and `fprintf()`.
5. Section: "Basic Math" — Arithmetic operators (+, -, *, /, ^). Example: `age_diff = ravn_age - helena_age`. Show that Matlab is a calculator.
6. Section: "Your First Collection: Vectors" — Introduce row vectors with `cat_weights = [4.2, 3.8, 5.1, 3.5, 4.7]`. Explain bracket notation. Show `length()`. Mention this is a preview — Module 02 goes deep on vectors.
7. Section: "Checkpoint" — "Run demo_hello.m and demo_variables.m. You should see a plot and printed text. Then try the exercise."
8. Troubleshooting box: script not found (wrong directory), variable not defined (forgot to run earlier lines).

Use short paragraphs (2-3 sentences max). Use analogies from Christel's life. NO math notation in this module — pure code and words.

**01-first-steps/demo_hello.m** — First script. Must produce a plot within first 5 executable lines:
```
clear; clc; close all;
x = 1:10;
y = x.^2;
plot(x, y, 'ro-', 'LineWidth', 2);
title('Your First Plot - You Made This!');
xlabel('x'); ylabel('x squared');
grid on;
```
Then after the plot, add commented explanations of each line. Add a second section (`%%`) that creates a different plot (e.g., bar chart of family ages) to show variety.

**01-first-steps/demo_variables.m** — Variables demo. Start with `clear; clc; close all;`. Create variables using Christel's context:
- Scalars: enzo_weight, ravn_age, helena_age
- String: cat_name = 'Enzo'
- Boolean: is_cat = true
- Arithmetic: age_diff, total_age, enzo_weight_pounds
- fprintf output for each
- A vector preview: cat_weights = [4.2, 3.8, 5.1, 3.5, 4.7], then bar(cat_weights) with labels
- Use %% section breaks between logical chunks
- Every section should have a disp() or fprintf() showing what happened

Follow the standard script header template from research:
```

%

%

```
