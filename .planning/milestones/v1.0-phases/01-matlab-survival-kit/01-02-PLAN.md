**02-vectors-and-plotting/lesson.md** — Written lesson. Structure:
1. Opening: "In this module you will learn to work with collections of numbers (vectors and matrices) and turn them into beautiful plots."
2. Section: "Vectors — Ordered Lists of Numbers" — Analogy: "A vector is like a shopping list — an ordered collection. The weights of 5 cats at the vet: `cat_weights = [4.2, 3.8, 5.1, 3.5, 4.7]`." Cover bracket notation, colon operator (`1:10`, `0:0.1:2*pi`), `linspace()`. Show row vs column vectors (`[1 2 3]` vs `[1; 2; 3]`) with `size()` to check. Mention transpose `v'`.
3. Section: "Grabbing Pieces — Indexing and Slicing" — Cover `v(1)` (1-based!), `v(3)`, `v(1:3)`, `v(end)`, `v(end-2:end)`. Explicitly note: "Matlab starts counting at 1, not 0." Matrix indexing: `M(2,3)`, `M(2,:)`, `M(:,1)`. Use the cat_weights vector for all examples.
4. Section: "Matrices — Tables of Numbers" — Analogy: "A matrix is like a spreadsheet — rows and columns." Create a 3x3 matrix of garden plot measurements. Show `zeros()`, `ones()`, `rand()`, `size()`.
5. Section: "Element-wise Operations" — Explain the dot prefix rule: `.* ./ .^` mean "do it to each element." Demo `v.^2` vs `v^2` (which errors for a non-square). Show `v1 .* v2` element by element.
6. Section: "Making Plots" — Walk through all plot types with one example each: `plot()` (line), `scatter()` (dots), `bar()` (bars), `imagesc()` (heatmap). Cover `xlabel()`, `ylabel()`, `title()`, `legend()`, `grid on`, `hold on/off`, `figure()`, `subplot()`. Reference demo_plotting.m.
7. Section: "Vectors as Arrows — Your First Math Concept" — THIS IS THE NOTATION BRIDGE. Order is critical:
   a. Show a picture description: "Imagine standing at the origin of a map. A vector tells you where to walk: 3 steps east and 4 steps up."
   b. Code: `v = [3, 4]` and `quiver(0, 0, 3, 4, 0)` — "Run demo_vector_arrows.m to see this"
   c. Direction: "The arrow points northeast — that is the direction"
   d. Magnitude: "The arrow is 5 steps long — that is the magnitude. In Matlab: `norm(v)` gives 5"
   e. ONLY NOW introduce notation: "Mathematicians write the length as ||v|| = sqrt(3^2 + 4^2) = 5"
   f. Explain: "You already know this — it is Pythagoras! The vector is the hypotenuse."
8. Section: "Checkpoint" — "Run all three demo scripts. Then try the exercises."

Keep short paragraphs. Use analogies from Christel's life (cats, garden, family).
